ESPHome On.e

Pilotez votre module On.e de piscine depuis Home Assistant via un ESP32 et ESPHome. Pompe, eclairage, en WiFi.

Voir sur GitHub Outil BLE

Fonctionnalites

💧

Pompe

Allumez et eteignez la pompe de filtration depuis Home Assistant ou l'interface web locale.

💡

Eclairage

Controlez l'eclairage de votre piscine, y compris depuis des automatisations HA.

📶

Retour d'etat

Statut en temps reel via notifications BLE. Vous savez toujours si la pompe ou la lumiere est active.

🔒

Authentification

Protocole challenge-response AES-128-ECB entierement implemente. Votre piscine reste securisee.

🕰

Horloge RTC

Synchronisation automatique de l'horloge du module On.e a chaque connexion.

🌐

100% local

Aucun cloud necessaire. Tout fonctionne en local entre l'ESP32, le module BLE et Home Assistant.

Installation rapide

Recuperer la SharedKey

Ouvrez l'outil BLE sur Chrome Android, mettez votre module en mode appairage, et scannez. La SharedKey s'affiche automatiquement.

Configurer ESPHome

Editez secrets.yaml avec votre WiFi, l'adresse MAC du module et la SharedKey recuperee.

Flasher l'ESP32

Lancez esphome run one-pool.yaml pour compiler et flasher votre ESP32.

Profiter dans Home Assistant

Le device apparait automatiquement avec les switches Pompe et Lumiere. Creez vos automatisations !
