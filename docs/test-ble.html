⚠️ Premiere utilisation

Pour recuperer la SharedKey de votre module, vous devez d'abord le mettre en mode appairage :

  1. Appuyez sur le bouton physique du module On.e dans votre tableau electrique jusqu'a ce que le voyant clignote
  2. Cliquez sur Connecter ci-dessous et selectionnez votre device
  3. La SharedKey s'affiche automatiquement en bas de page — notez-la
  4. Vous pouvez ensuite tester les controles (pompe / lumiere)

💡 La SharedKey ne doit etre recuperee qu'une seule fois. Elle est sauvegardee dans le navigateur.
